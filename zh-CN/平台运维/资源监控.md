资源监控功能提供了对平台运行环境关键系统指标的实时监控，帮助用户了解系统资源使用情况。

![](./image/resource_stat.png)

|  监控指标| 说明|
| -------- | -------------------------------- |
| jvm内存使用率 | 通过内存管理MXBean获取jvm已用堆内存和最大可用堆内存计算 |
| 磁盘IO使用率  | 通过iostat指令获取，为YMP部署所在磁盘的IO使用率    |
| CPU使用率   | 通过top指令获取，为YMP部署所在环境CPU使用率       |
| CPU核数    | 平台部署所在环境CPU核数                    |

> **Note**:
> 
> 当部署在容器内部时，监控的资源为宿主机资源而非容器内部资源。
