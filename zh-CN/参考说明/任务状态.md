YMP中评估、迁移、校验都是依托任务来进行的，本章节介绍任务各个状态。  

### 创建任务-成功

创建任务界面保存任务配置后，任务生成，状态为成功。

在当前状态下，可进行的操作有：  

|  可选操作| 操作后下一个任务状态| 操作说明|
|--------------------| -------------------------- | ------------------------ |
| 保存任务             |    创建任务-成功         | 保存任务，状态不变 |
| （开始步骤为迁移的任务）迁移配置   |    创建任务-获取对象中 | |
| 开始迁移评估         |   迁移评估-评估中 | |
| 删除任务            |   任务被删除，无状态 | |
