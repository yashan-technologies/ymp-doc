

YMP支持以下组合类型的迁移操作，您可按需创建任务：

|  迁移步骤| 说明|
| -------- | -------------------------------------- |
| 评估       | 只进行评估操作，可选在线评估或者离线评估           |
| 迁移       | 只进行迁移操作，未进行数据库对象兼容性评估，迁移过程中可能出现不兼容现象   |
| 校验       | 只进行校验操作                                |
| 在线评估+迁移  | 进行评估+迁移操作                              |
| 迁移+校验    | 进行迁移+校验操作，未进行数据库对象兼容性评估，迁移过程中可能出现不兼容现象 |
| 在线评估+迁移+校验 | 默认步骤，迁移全流程操作                           |
| 配置参数       | 可以与上述步骤任意组合，进行配置参数迁移操作                   |

其中，在线评估与离线评估的主要区别为，离线评估结果不能被用于后续的迁移阶段，因此对需要执行迁移的任务，应该选择在线评估。

以下为创建迁移任务时的输入项目说明：

::: tabs

== 在线评估

|  字段| 说明|
| ------------ | ------------------------------------------------ |
| 任务名称     | 不能与现有任务名称重复                           |
| 迁移步骤     | 支持完整的迁移全流程操作，可按需选择             |
| 评估类型     | 在线评估，默认评估方式，建议源端保持静默离线状态 |
| 源端数据库   | 选择DM                                           |
| 目标端数据库 | 选择YashanDB                                     |
| 备注说明     | 该任务的简要说明                                 |

== 离线评估

|  字段| 说明|
| -------- | ---------------------------- |
| 任务名称 | 不能与现有任务名称重复       |
| 迁移步骤 | 只支持评估，不支持迁移和校验 |
| 评估类型 | 离线评估                     |
| 源端类型 | 选择DM                       |
| 备注说明 | 该任务的简要说明             |

:::

【创建任务】和【下一步：继续配置】都会检查源端数据库连接用户是否满足选择的任务步骤所需的权限，缺失的权限会弹窗提示。
