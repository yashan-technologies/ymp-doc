

校验阶段用于对源端和目标端的数据进行是否一致性检查，YMP提供了全量校验、主键校验和统计校验三种模式，根据数据量和业务重要性，可采用不同的校验策略：

|  数据量| 业务重要性| 推荐校验策略|
| --------------- | ---------- | ------------ |
| 小（<100GB）    | 高         | 全量校验     |
| 小（<100GB）    | 低         | 全量校验     |
| 中（100GB-1TB） | 高         | 全量校验     |
| 中（100GB-1TB） | 低         | 统计校验     |
| 大（>1TB）      | 高         | 全量校验     |
| 大（>1TB）      | 低         | 统计校验     |

一致性校验主要用于全量迁移后的数据检查，用户也可以创建一个独立的校验任务，两者间的区别主要在于校验的可选对象范围。

YMP为一致性校验过程中所需的各项配置、监控、异常处理等用户接口均提供了界面化交互指引，用户也可依据如下文档指导进行操作：

- [校验初始化](校验初始化)：校验初始化阶段为校验任务的准备阶段，主要完成校验任务的配置初始化。
- [一致性校验](一致性校验)：依据配置执行源端和目标端一致性校验。
