## Overview

The YashanDB Migration Platform (YMP) is a database migration product provided by YashanDB. It offers the capabilities of assessment before migration, data migration, and data validation between heterogeneous RDBMS and YashanDB.  
YMP provides a visual service, allowing users to complete the entire process from assessment to migration through simple interface operations, achieving low threshold, low cost, and high efficiency in heterogeneous database migration.

## Core Features

### Accurate Assessment

Based on a pluggable database minimum kernel, SQL compatibility is accurately verified during the assessment phase, resulting in trustworthy evaluation results.

### SQL Auto-Convert

Supports automatic conversion of SQL statements, fully adapting to YashanDB syntax, resolving most compatibility issues related to SQL adaptation, and significantly reducing manual adaptation workload.

### Seamless Integration of Assessment and Migration

The assessment results automatically serve as the basis for migration. During migration, object DDL is rewritten automatically or manually according to what was done in the assessment phase, ensuring that compatibility in assessment translates to successful migration.

### One-Click Object Migration

Integrates all object metadata migration in one click, fully considering object dependencies and their impact on data migration, and employs an end-to-end performance-optimized execution strategy that intelligently decides the creation order of metadata objects.

### High-Performance Data Migration

Utilizes the database's native high-performance import and export capabilities and adopts a multi-level parallel architecture to achieve factory-level high-performance data migration.

### Data Validation

Utilizes the database's own query and sorting capabilities to concurrently extract data, processes heterogeneous data sources into a unified format for comparison, ensuring data consistency.

### One-Stop Service

Integrates the entire migration process, including assessment before migration, data migration, and data validation, on one platform, providing users with a low-threshold, visual, and replicable one-stop migration service.

## Application Scenarios

YMP is an offline assessment migration tool designed for database migration scenarios, addressing compatibility and workload estimation issues, heterogeneous database metadata migration, and fast data migration challenges.
