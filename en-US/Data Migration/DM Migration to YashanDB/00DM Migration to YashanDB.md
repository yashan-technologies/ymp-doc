Database migration is a systematic project that requires thorough preparation and rigorous execution. Please read the operation guide before migration to understand the YMP migration process and related specification constraints. After the migration, fully utilize the advantages of YashanDB to enhance your database performance and reliability. If you encounter any unanswered questions in the documentation during the migration process, please contact our technical support for assistance.

When migrating the DM database from YMP to YashanDB, please refer to [Product Specifications](../../Product Introduction/Specifications) for supported database versions, database objects, and data type specifications.

The target end of YashanDB can only be in yashan mode.
