Database migration is a systematic project that requires adequate preparation and rigorous execution. Please read the operation guide before migration to understand the YMP migration process and relevant specifications and constraints. After migration, fully utilize the advantages of YashanDB to enhance your database performance and reliability. If you encounter any unanswered questions during the migration process, please contact our technical support for assistance.

For the migration of Oracle databases to YashanDB via YMP, the supported database versions, database objects, and data types can be found in the [Product Specifications](../../Product Introduction/Specifications).

The target end of YashanDB can only be in the yashan mode.
