The assessment phase is used to check the compatibility of metadata between the source and target ends. YMP conducts compatibility assessments of heterogeneous databases through object dimensions. During the assessment process, the DDL statements for creating objects may be modified to a certain extent.

YMP provides two assessment methods: online assessment and offline assessment. Among these, the metadata from online assessments can serve as input for source information in the subsequent migration phase, while offline assessments can only be used to execute assessment tasks.

When using online assessments, it is recommended that the source end remain in a silent offline state to ensure data consistency.
