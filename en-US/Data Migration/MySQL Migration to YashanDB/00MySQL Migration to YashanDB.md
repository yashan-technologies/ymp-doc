Database migration is a systematic project that requires thorough preparation and rigorous execution. Please read the operation guidelines before migration to understand the migration process of YMP and related specification constraints, and fully utilize the advantages of YashanDB to enhance the performance and reliability of your database after migration. If you encounter any unresolved issues during the migration process, please contact our technical support for assistance.

For the specifications regarding the supported database versions, database objects, and data types when migrating MySQL databases to YashanDB, please refer to the [Product Specifications](../../Product Introduction/Specifications).

The target end of YashanDB can be in yashan mode or mysql mode.
