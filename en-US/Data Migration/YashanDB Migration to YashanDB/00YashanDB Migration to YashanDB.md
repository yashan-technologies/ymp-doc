Database migration is a systematic project that requires adequate preparation and rigorous execution. Please read the operation guide before migration to understand the YMP migration process and relevant specification constraints, and make full use of YashanDB's advantages to improve your database performance and reliability after migration. If you encounter questions not answered in the documentation during the migration process, please contact our technical support for assistance.

For the specifications regarding supported database versions, database objects, and data types when migrating YashanDB to YashanDB using YMP, please refer to the [Product Specifications](../../Product Introduction/Specifications).

The source and target ends of YashanDB can only be in yashan mode.
