Database migration is a systemic project that requires thorough preparation and rigorous execution. Please read the operation guidelines before migration to understand the YMP migration process and related specification constraints, and fully utilize the advantages of YashanDB to enhance your database performance and reliability after migration. If you encounter questions during the migration process that are not addressed in the documentation, please contact our technical support for assistance.

When migrating the DB2 database to YashanDB using YMP, please refer to the [Product Specifications](../../Product Introduction/Specifications) for supported database versions, database objects, and data types.

The target endpoint for YashanDB can only be in yashan mode.
