This section mainly provides the overall process operations for user management, report management, and security management.

|Functionality Module |Description |
| ----------------- | ------------------------------------------------------- |
| [User Management](User Management) | YMP provides an initial administrator account for logging into the system. Users are required to change their password upon first login. |
| [Data Source Management](Data Source Management) | YMP provides management for all databases involved in YMP services, including: source database, target database, assessment database. |
| [Report Management](Report Management) | Corresponding reports are generated at the end of each phase of YMP tasks, which can be managed more conveniently through the report management page. |
| [Security Management](Security Management) | YMP supports enabling HTTPS to protect data transmission between the client (e.g., browser) and the server through encryption and authentication. |
| [Resource Monitoring](Resource Monitoring) | YMP provides monitoring capabilities for JVM memory, disk IO, and CPU. |
