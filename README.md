ymp-doc是深圳崖山科技有限公司的崖山迁移平台（YashanDB Migration Platform，YMP）的产品文档仓库，本仓库内容将定期与[YashanDB官网文档中心](https://doc.yashandb.com/)内容进行同步。

YashanDB产品下载中心：[https://download.yashandb.com](https://download.yashandb.com)

## 文档结构

每级目录下包含00文件（必选）、_index文件（可选）、image目录（可选）、子目录及文档文件：

1. 00文件：名称为00+目录名，内容为目录整体介绍或前言导引。
2. _index文件：控制文档在官网上的导航顺序，initno项目为预留项，filename项目指定以,分隔的各子目/文件名称，enName为与filename一一对应的英文名称。当目录下为全英文命名的子目录/文件，且希望以默认字母排序导航时，本文件可省略。
3. image目录：包含当前目录下所有子文件需要应用的图片，必须为png图片。图片以./image/图片名称格式进行引用。
4. 文档文件：遵循通用Markdown格式书写。
